AI Agent Resources

This page provides resources for AI agents implementing multi-agent orchestration patterns. These files follow the llms.txt specification and related patterns for AI-readable content.

Skill Files

The following markdown files provide instructions and patterns for AI agents:

File Purpose
RLM.md Recursive Language Models - actionable instructions for task decomposition and handling large contexts
RLM-extra.md Detailed RLM reference - architecture, benchmarks, prompt templates, error handling
MULTI-AGENT.md AI agent orchestration patterns - how to coordinate multiple sub-agents for complex tasks
CLAUDE-CODE.md Using Claude Code CLI as a sub-agent - full tool access, web search, parallel execution
CODEX.md Using Codex CLI as a sub-agent - ideal for image analysis, focused code review, and parallel workloads
GEMINI.md Using Gemini CLI as a sub-agent - provides alternative perspective for cross-validation and long-context tasks

How to Use

Copy these files to your project root to provide AI agents with orchestration patterns:

# Download skill files to your project
curl -sO https://example.com/RLM.md
curl -sO https://example.com/MULTI-AGENT.md
curl -sO https://example.com/CLAUDE-CODE.md
curl -sO https://example.com/CODEX.md
curl -sO https://example.com/GEMINI.md

Or reference them in your project’s CLAUDE.md or similar instruction file:
