AI Agent Resources

This site provides resources for AI agents to better understand and interact with the content. These files follow the llms.txt specification and related patterns for AI-readable content.

Skill Files

The following markdown files provide instructions and patterns for AI agents working with this repository or implementing multi-agent orchestration:

File Purpose
RLM.md Recursive Language Models implementation guide - theoretical foundation for task decomposition and handling inputs beyond context windows
MULTI-AGENT.md AI agent orchestration patterns - how to coordinate multiple sub-agents for complex tasks
CLAUDE-CODE.md Using Claude Code CLI as a sub-agent - full tool access, web search, parallel execution
CODEX.md Using Codex CLI as a sub-agent - ideal for image analysis, focused code review, and parallel workloads
GEMINI.md Using Gemini CLI as a sub-agent - provides alternative perspective for cross-validation and long-context tasks

Blog Post Markdown

All blog posts written in markdown are also available in their raw form for AI agents. Append index.md to any blog post URL to get clean markdown without HTML rendering:

https://jonnyzzz.com/blog/2024/09/17/Eugene-20-years/          → HTML page
https://jonnyzzz.com/blog/2024/09/17/Eugene-20-years/index.md  → Clean markdown

The .md files contain no YAML front matter. Metadata is provided as readable text at the top of each file.

How to Fetch

AI agents can fetch these resources using curl or any HTTP client:

# Fetch skill files
curl -s https://jonnyzzz.com/RLM.md
curl -s https://jonnyzzz.com/MULTI-AGENT.md
curl -s https://jonnyzzz.com/CLAUDE-CODE.md
curl -s https://jonnyzzz.com/CODEX.md
curl -s https://jonnyzzz.com/GEMINI.md

# Fetch a blog post as markdown
curl -s https://jonnyzzz.com/blog/2024/09/17/Eugene-20-years/index.md

# Pipe to your context or save to file
curl -s https://jonnyzzz.com/RLM.md | head -100
curl -s https://jonnyzzz.com/RLM.md > /tmp/rlm-context.md

For AI agents with web fetch capabilities:

WebFetch: https://jonnyzzz.com/RLM.md
WebFetch: https://jonnyzzz.com/blog/2024/09/17/Eugene-20-years/index.md

Usage
